# JSP中Tag相关内容总结

今天把`JSP`书中Tag章节看完了，把书上的例子都实现了一遍。
下面回顾下Tag中最为重要的内容。
1. 为什么会有Tag文件？

在使用Tag文件之前，我们已经可以在jsp页面中包含其他jsp页面（两种方式）,但是这种方式存在一个大缺点，我们无法避免用于直接输入地址访问
我们要包含的jsp页面。如果要包含的页面只是一个导航条，那么用户访问是无意义的。那么有没有办法可以避免用于直接访问我们的不想被看见的
内容，那就是使用tag文件。

2. Tag文件与jsp文件异同

其实大部分都是相同的，都是实现复用代码。

3. Tag文件中比较核心的功能

传递参数，返回结果。对应attribute、variable属性。
有了这两个功能，Tag文件的功能就十分强大了。

4. Tag文件中中文乱码问题解决

加上一行`<%@tag pageEncoding="UTF-8" %>`。这样就显示正常了。
关于乱码问题，会有一篇博文专门研究。
